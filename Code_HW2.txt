For part 4, I wrote the script hw2_client.py, which is in the GitHub repository. The script follows the directions given in the homework 2 description and interacts with both of the VMS. The code measures the latency for the /register and /list endpoints on both instances. The script sends 10 requests and records the elapsed time per request using time.perf_counter(). The average latency is then given in milliseconds. /register the request registers unique usernames to avoid key collisions and /list retrieves the json list of usernmaes in the database. For observing consistency, as the description said, the code runs 100 iterations to test cross-region visibility. It sends a /register request on the US vm and sends a /list request on the Europe VM. Next, it checks whether the username appears on the returned list. Then we count the total number of misses and reported that at the very end.
